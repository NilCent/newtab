# 苦练ing

一个可定制的 Chrome 新标签页扩展, 集成待办事项, 便签, Flashcard, 定时器等组件.

## 为什么开发这个扩展

我之前使用各种效率工具时遇到一个共通问题: **工具无法强制我使用它们**. 手机上的 Flashcard 和番茄钟可能几个月都没打开过.

痛点的核心是: **如何让自己每天多次看到这些工具**.

答案就是浏览器的新标签页——每次打开新标签页都会看到这些组件. 页面参考了 iPhone 小组件的设计, 可以直接在页面上查看和编辑, 无需打开独立应用.

---

## 主要组件

### 待办事项 (MustDone)

**痛点**: 之前使用苹果的提醒事项, 遇到三个问题:
- 无法与 Windows 同步
- 提醒经常被忽略
- 设置重复任务比较复杂

**解决方案**:
- 通过 Google 账号同步, 跨设备使用
- **惩罚机制**: 任务逾期会禁用历史记录和搜索栏, 强制完成任务
- 简化重复规则: 支持每日, 每周指定日期
- 支持计数任务: 如「每周日前完成 7 道算法题」

---

### 便签

**痛点**: 之前使用苹果备忘录:
- 不支持 Markdown, 难以正经写笔记
- 无法同步到 Windows

**解决方案**:
- 通过 Google 账号同步
- 最多显示约 10 行内容, 避免堆积过多想法
- 适合记录书名, 知识点等随手记

---

### Flashcard

**痛点**: 之前使用 RemNote, 笔记整理完还得再整理 Flashcard, 流程割裂.

**解决方案**: 扩展 Markdown 语法, 在笔记中直接创建卡片:

```markdown
# 单行双面卡片
正面 >>> 反面

# 多行双面卡片
正面内容
:::
反面内容
可以多行
:::

# 单面卡片(用 --- 分隔)
第一个知识点
---
第二个知识点
```

---

### 定时器

**痛点**: 之前使用 Forest, 只能在手机上使用.

**解决方案**: 简单实用的定时器, 计时结束发送系统通知.

---

### 历史记录

**痛点**: 之前使用 Infinity 新标签页, 但没有黑名单功能.

**解决方案**:
- 直接在新标签页搜索历史记录
- 支持黑名单过滤敏感内容

---

### Hacker News

**痛点**: 总是懒得打开 Hacker News 网页, 错过了很多技术前沿信息.

**解决方案**: 在新标签页直接展示热门文章, 逼自己获取技术前沿动态.

---

### 阅读列表

**痛点**: 平时用 Chrome 阅读列表收集有价值的技术文章, 但总是收藏后就不再打开.

**解决方案**: 在新标签页展示阅读列表, 提醒自己看看收藏的内容.

---

## 安装

1. 克隆仓库并安装依赖:
   ```bash
   git clone https://github.com/your-repo/newtab.git
   cd newtab
   npm install
   ```

2. 构建扩展:
   ```bash
   npm run build
   cp manifest.json dist/
   ```

3. 在 Chrome 中加载:
   - 打开 `chrome://extensions/`
   - 开启「开发者模式」
   - 点击「加载已解压的扩展程序」
   - 选择项目的 `dist` 目录

---

## 自定义布局

- 点击右下角 ⚙ 按钮进入编辑模式
- 拖拽组件调整位置
- 点击 + 按钮添加新组件
- 点击组件右上角 − 按钮删除

---

## 数据同步

所有数据通过 Chrome Storage API 同步, 登录 Google 账号后可在不同设备间同步:
- 待办事项
- 便签内容
