# NewTab

一个用于提升学习效率的 Chrome 新标签页扩展。

## 背景

在使用过各种效率工具（Flashcard、看板、OKR、番茄钟等）后，我发现一个痛点：这些工具无法**强制**我使用它们，最多只是发几个提醒。

我的解决方案：把这些工具放到 newtab 里。因为我每天访问最多的就是 Chrome 的新标签页，通常从这里进入其他页面。

**核心机制**：如果有待办事项逾期未完成，就禁止使用搜索栏和历史记录。

## 功能

### 待办事项
- 页面上显示当前还未完成的任务
- 任务逾期时强制锁定搜索栏和历史记录
- 支持进度追踪（如：完成 3/5 道题）

### 闪卡
- 扩展 Markdown 语法, 使其支持 flashcard
- 支持单面卡片和双面卡片（点击翻转）
- 从本地目录或单个 Markdown 文件加载

### 其他组件
- **翻译** - 多语言翻译面板
- **天气** - 7 天天气预报
- **日期时间** - 显示当前时间
- **Hacker News** - 热门技术文章
- **历史记录** - 最近访问的链接


## 安装

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```

构建后, 将 `manifest.json` 放到 `dist` 目录中, 然后将 `dist` 目录加载到 Chrome 扩展程序中（开发者模式）。


## 闪卡 Markdown 语法

```markdown
# 单行双面卡片
正面 >>> 反面

# 多行双面卡片
正面内容
:::
反面内容
可以多行
:::

# 单面卡片（用 --- 分隔）
第一个知识点
---
第二个知识点
---
第三个知识点

# 标题作为标签
# 英语单词
immune >>> 免疫的
```
